Pandemie Game

This project is an digital version of the Pandemic Board Game with its original rules using native Java. This project was done in collaboration with two friends for a school project.
My most notable contributions were the implementation of:

  • underlying graph data structure using a HashSet
  • responsive UI with zoom, scroll, layout and coloring
  • card shuffling algorithm
  • breath first search algorithm

Rubik's Cube solver

This project is a solver of the popular 3x3x3 Rubik's Cube from any given position using native Haskell.
This was achieved by encoding all states into an array of fixed positions of every single 28 induvidual cubes. While eradicating most symmetrical positions the self-implemented CDCL algorithm will try to find the solution.
